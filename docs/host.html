Ik wil op mijn community board van mijn vossenjachtspel dat de knop voor volgende fase een andere kleur krijgt zodat we zien wanneer alle spelers klaar zijn met een ronde. De engie manier waarop ik het nu kan zien is omdat de spelerskaarten knipperen bij iedere beurt. Wanneer dit knipperen ophoudt klik ik op volgende fase en kunnen we verder. TIjdens de action fase blijven de spelers kaarten knipperen zelfs nadat alle spelers PASS heeft gekozen. Ik wil dat de volgende fase knop een indicatie geeft wanneer ik deze kan indrukken. We kunnen een optie toevoegen Raid pauzeren aan het einde van iedere fase. Liever wil ik dat de volgende fase automatische geactiveerd wordt na een pauze van 3 seconden, mits ik een pauzeknop indruk.
